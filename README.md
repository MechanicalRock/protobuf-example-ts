# Getting Started with Protobufs and Typescript
See Mechanical Rock blog post [Protobufs for TypeScript]()

## To use this repository
1) Install `protoc` as per blog post instructions above
2) Run `npm install`
2) Run `./compileProtos.sh`
3) Run `npm run test`
